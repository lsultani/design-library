# Typography

Two typefaces. One voice.

**Space Grotesk** carries the site — headlines, body, and UI labels.
Weight range is intentionally narrow (Light, Regular, Medium, Bold).
**Space Mono** is used as a meta face: eyebrows, tags, nav chips,
timeline dates, chart labels. Never body copy.

## Rules

- Headlines never use uppercase. Ever.
- Mono is always uppercase with at least 0.15em tracking.
- Don't add weights beyond Light/Regular/Medium/Bold for Space Grotesk.
- Don't use italic except in pull quotes.
